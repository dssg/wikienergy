Generators¶

The generate module contains various methods for generating new appliances and appliance traces.

Methods¶

disaggregator.generate.generate_instance(starts, periods, freq)[source]¶

Returns a randomly generated appliance instance for a particular time period.

disaggregator.generate.generate_refrigerator_trace(start, periods, freq)[source]¶

Uses hmm parameters learned from Pecan Street data to generate samples for a fairly normal cycling refrigerator.

Uses a three state HMM.

disaggregator.generate.generate_trace(start, periods, freq)[source]¶

Returns a randomly generated appliance trace for a particular time period. Ex) generate_trace(datetime.datetime(2013,1,1),96,‘15T’))
